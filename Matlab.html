PreProccessing

The first step to calculate flame height is to convert the raw image of a Flame to a suitable format for the code

The raw image of the flame as it can be seen is in color format and has other information , so it is neccessary to first identify the flame. To do so we remove all the static parts of the image , by selecting a frame in the movie as the refrence frame and subtracting all the images from the constant image

the first step in flame identification is converting the color format "RGB" to Grayscale this is done by Matlab function

GRAY=rgb2gray(PIC)

in the next step we define the brighter location of the image to be related to the fire, this can be done by setting a level . Any point higher than the level will be converted to 1 , and any intensity lower than the level will be zero. The Matlab function for thi step is

BW=im2bw(PIC,level)

#NOTE# the level shoud be a value between 0 and 1 , and it highly depends on the flame images. the default value of the level in Matlab is 0.5
